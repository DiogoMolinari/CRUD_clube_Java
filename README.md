☕ Sistema de Gestão de Clube Esportivo
Este projeto consiste em um sistema desenvolvido em Java com o objetivo de gerenciar os dados de um clube esportivo, permitindo o controle completo de sócios, planos, pagamentos e feedbacks.
O sistema foi criado como parte dos estudos em programação, com foco na aplicação prática dos conceitos de Programação Orientada a Objetos e operações de CRUD.


⚙️ Funcionalidades
O sistema implementa operações completas de CRUD (Create, Read, Update, Delete) para as seguintes entidades:
👤Sócios
  * Cadastro de novos membros
  * Consulta de dados
  * Atualização de informações
  * Remoção de registros

  💳 Formas de pagamento
  * Cadastro e gerenciamento de métodos de pagamento

 📋 Planos
  * Criação e edição de planos disponíveis no clube

  💬 Feedbacks
  * Registro de opiniões e avaliações dos sócios


🛠️ Tecnologias utilizadas
* Java


📚 Conceitos aplicados
* Programação Orientada a Objetos (POO)
* Encapsulamento
* Estruturação de sistemas em camadas
* Manipulação de dados (CRUD)
* Lógica de programação

🎯 Objetivo do projeto
Aplicar na prática os conceitos fundamentais de desenvolvimento de software, simulando um sistema real de gerenciamento utilizado em clubes esportivos.
